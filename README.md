# 🐍🎲 সাপলুডু ও লুডু — Saplu-Ludu Game

বাংলা ভাষায় তৈরি দুটি ক্লাসিক বোর্ড গেম — **সাপলুডু (Snake & Ladder)** এবং **লুডু (Ludo)** — একসাথে একটি repo-তে। বিশুদ্ধ HTML/CSS/JavaScript দিয়ে তৈরি (কোনো ফ্রেমওয়ার্ক বা বিল্ড-স্টেপ লাগে না), যাতে পরে Termux + GitHub Actions workflow দিয়ে সহজেই Android APK বানানো যায় (Capacitor/Cordova দিয়ে)।

## 📁 ফোল্ডার স্ট্রাকচার

```
saplu-ludu-game/
├── www/                     ← এই পুরো ফোল্ডারটাই হলো অ্যাপের "web assets"
│   ├── index.html           ← হোম স্ক্রিন (গেম বাছাই)
│   ├── manifest.json        ← PWA manifest (অ্যাপ আইকন, নাম, থিম কালার)
│   ├── css/
│   │   └── common.css       ← শেয়ার্ড থিম/ডিজাইন টোকেন
│   ├── assets/
│   │   └── icon.svg         ← অ্যাপ আইকন
│   ├── saplu/                ← সাপলুডু গেম
│   │   ├── index.html
│   │   ├── style.css
│   │   └── script.js
│   └── ludo/                ← লুডু গেম
│       ├── index.html
│       ├── style.css
│       └── script.js
├── package.json
├── .gitignore
└── README.md
```

`www/` ফোল্ডারটিকে এমনভাবে সাজানো হয়েছে যাতে এটি সরাসরি **Capacitor** বা **Cordova**-এর `webDir`/`www` হিসেবে ব্যবহার করা যায় — পরের ধাপে APK বানানোর সময় এটাই কাজে লাগবে।

## 🎮 ফিচার

### সাপলুডু (Snake & Ladder)
- ক্লাসিক ১০×১০ (১–১০০) বোর্ড, জিগজ্যাগ নাম্বারিং
- ২ থেকে ৪ জন প্লেয়ার (হট-সিট মোড, একই ডিভাইসে পালা করে খেলা)
- ৮টি সাপ, ৮টি মই — প্রতিটিতে ইমোজি ও দিক-নির্দেশক
- ডাইস রোল অ্যানিমেশন, টোকেন স্লাইড/ক্লাইম্ব অ্যানিমেশন
- ৬ পড়লে আবার চান্স, জয়ী হলে সুন্দর সেলিব্রেশন মোডাল

### লুডু (Ludo)
- পূর্ণাঙ্গ ক্লাসিক লুডু বোর্ড (লাল-হলুদ-সবুজ-নীল, ৫২ ঘরের ট্র্যাক + হোম কলাম)
- ২–৪ জন প্লেয়ার, প্রতি প্লেয়ারের ৪টি গুটি
- গুটি কাটাকাটি (capture), সেফ/স্টার ঘর, ৬ পড়লে এক্সট্রা চান্স
- কোন গুটি চলতে পারে তা হাইলাইট করে দেখানো, গুটিতে ট্যাপ করে মুভ
- সব গুটি ঘরে পৌঁছালে জয়ী ঘোষণা

## 🎨 ডিজাইন

মাটির রঙের (terracotta/cream) ঐতিহ্যবাহী বাংলা বোর্ড-গেমের আমেজে তৈরি — গাঢ় ইন্ডিগো ব্যাকগ্রাউন্ড, হাতে-আঁকা প্যাটার্নের ছোঁয়া, এবং লাল-সবুজ-হলুদ-নীল গুটির উজ্জ্বল রঙ। ফন্ট হিসেবে ব্যবহৃত হয়েছে Google Fonts-এর **Baloo Da 2** (হেডিং) ও **Noto Sans Bengali** (লেখা), যা ইন্টারনেট থেকে CDN লিংকে লোড হয়।

## ▶️ লোকালি চালানো

ব্রাউজারে সরাসরি `www/index.html` খুললেই চলবে, অথবা যেকোনো সিম্পল লোকাল সার্ভার দিয়ে:

```bash
cd www
python3 -m http.server 8080
# তারপর ব্রাউজারে http://localhost:8080 খুলুন
```

## 📱 অ্যাপ বানানো (পরের ধাপ)

এই repo-টা যেভাবে সাজানো হয়েছে, তা Capacitor/Cordova + GitHub Actions workflow দিয়ে Android APK বিল্ড করার জন্য রেডি। সেই কমান্ডগুলো এবং workflow ফাইল পরের ধাপে যুক্ত করা হবে।
